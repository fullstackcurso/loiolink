Añadir vía repositorio

  1. En Kodi: Ajustes → Explorador de archivos → Añadir fuente.
  2. En <Ninguno> introducir esta dirección exacta:
    https://loiolo.io/
  3. Asignar el nombre loiolink, lo, o lo que más te guste y dale a aceptar.
  4. En Ajustes → Add-ons → caja abierta (arriba a la izquierda) → Instalar desde un archivo zip.
    (Si aparece aviso de seguridad: Ajustes → Sistema → Add-ons → activar "Orígenes desconocidos").
  5. Seleccionar la fuente loiolink → repository.loiolink-1.0.0.zip.
  6. Cuando aparezca "Add-on instalado", ir a Instalar desde repositorio → loiolink Repository → Add-ons de programas → loiolink → Instalar.

Vía ZIP

Misma instalación pero descargando el zip del repositorio directamente, sin configurar fuente:

  1. Descargar repository.loiolink-1.0.0.zip con el enlace de arriba.
  2. En Kodi: Ajustes → Add-ons → caja abierta → Instalar desde un archivo zip → seleccionar el archivo.
  3. Continuar como en el paso 6 de arriba.
Si instalas el addon directo (plugin.program.loiolink-0.3.1.zip) sin el repositorio, no recibirás actualizaciones automáticas.
